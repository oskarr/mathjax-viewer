# mathjax-viewer
A page to view LaTeX equations. Forked from @saxarona's [mathjax-viewer](https://github.com/saxarona/mathjax-viewer), MIT license.
Try it out [here](https://naughty-kalam-a1f827.netlify.app/).

## Features
* Custom output colors
* Permalinks
* Automatic rendering
* Insertion buttons
* LaTeX cheat-sheet
